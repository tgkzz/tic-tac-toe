
# Tic-Tac-Toe

Tic-Tac-Toe is a paper-and-pencil game for two players who take turns marking the spaces in a three-by-three grid with X or O. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row is the winner. It is a solved game, with a forced draw assuming best play from both players. My program is written on Go
## Installation

Install my-project with git bash

```bash
  git clone https://github.com/tgkzz/tic-tac-toe.git
  cd tic-tac-toe
```
    
## Run 

Start the game. paste the command below, inside project folder

```bash
  go run .
```
